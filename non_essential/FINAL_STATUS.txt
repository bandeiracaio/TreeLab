================================================================================
EXTENSIBILITY
================================================================================

Adding new actions is easy:

1. Create a new class inheriting from BaseAction
2. Implement 5 required methods:
   - get_parameters()
   - validate()
   - execute()
   - suggest_columns()
   - to_python_code()
3. Register in treelab/app.py
4. Restart TreeLab

Example: Adding MinMaxScaler would take ~15 minutes.
