TO LAUNCH TREELAB
================================================================================

Open a terminal and run:

    cd TreeLab
    python run_treelab.py

Then open in your browser:

    http://127.0.0.1:8050

That's it! TreeLab will be running.

================================================================================

================================================================================

When you open the browser, you'll see:

1. HEADER
   - "TREELAB" title

   - Action selector dropdown

   - History panel showing all actions

================================================================================
TROUBLESHOOTING
================================================================================

If port 8050 is in use:
  Edit run_treelab.py and change: app.run(port=8051)

If modules not found:
  pip install -r requirements.txt

If browser doesn't open:
  Manually navigate to http://127.0.0.1:8050
  Or try http://localhost:8050

If action fails:
  - Read the error message carefully
  - Common issue: Need to do train/test split before modeling
  - Common issue: Selected wrong column types
