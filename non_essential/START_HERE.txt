ğŸ¯ FIRST TIME USER WORKFLOW
================================================================================

1. Launch TreeLab (see Quick Start above)

2. In the browser:
   - Look at the Data View tab
   - Click on Statistics tab
   - Try Distributions tab

3. Select your first action:
   - Choose "DropColumns" from dropdown
   - Select some columns (passenger_id, name, ticket, cabin)
   - Click "Execute Action"
   - See the result!

4. Create a checkpoint:
   - Type "My First Checkpoint" in the text box
   - Click "Save"
   - See it appear in History

5. Continue with more actions:
   - SimpleImputer â†’ fill missing values
   - OneHotEncoder â†’ encode categorical columns
   - StandardScaler â†’ scale numeric features
   - TrainTestSplit â†’ split the data

6. Switch to Modeling Mode:
   - Click "Modeling" button
   - Select "RandomForestClassifier"
   - Adjust parameters
   - Click "Execute Action"

7. View Results:
   - Go to "Model Results" tab
   - See accuracy, confusion matrix, feature importance

8. Export your work:
   - Click "Export Python Script"
   - Download the file
   - You can run it independently!

â“ TROUBLESHOOTING

Problem: "Port already in use"
Solution: python run_treelab.py --port 8051
          (or edit run_treelab.py and change port number)

Problem: "Module not found"
Solution: pip install -r requirements.txt

Problem: "Cannot find titanic.csv"
Solution: Make sure you're in the TreeLab/ directory

Problem: Browser won't connect
Solution: Try http://localhost:8050 instead of 127.0.0.1:8050
